# YAMH

## Overview

__YAMH__ - Yet Another Multiplatform Home - An open-source smart home control app built with Kotlin Multiplatform and Compose Multiplatform, deeply integrated with Google Home SDK.

## Features

- For now, the app supports Android only, which is enough to validate the core smart home use cases at the MVP stage
- Full support for portrait & landscape orientations, including foldable devices
- Home screen widgets, Quick Settings Tile Service, and Device Provider Service
- Google Home SDK integration
- Favorites and quick access to saved devices
- Custom theming
- Custom localization
